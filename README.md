## WithPath

내 하루의 이동을 기록하고, 필요한 사람에게만 안전하게 공유하는 동선 기록 앱입니다.

- 자동 동선 기록
- 장소별 체류 시간 확인
- 소중한 사람과 위치 공유
- 프라이버시 중심 설계
- 장소 저장 및 태그 관리
